Sistemas de medida en CSS

CSS utiliza distintos sistemas de medida para definir tamaños, espacios y ubicaciones. Estos pueden ser absolutos o relativos dependiendo del comportamiento deseado en el diseño.

Unidades Absolutas

Las unidades absolutas incluyen `px`, `cm`, `mm`, `in`, `pt`, y `pc`. Son fijas y no se adaptan a diferentes tamaños de pantalla o configuraciones de usuario.

Unidades Relativas

Las unidades relativas como `%`, `em`, `rem`, `vw`, y `vh` cambian en función del tamaño del contenedor, del texto raíz o del viewport. Son ideales para diseño responsivo.
