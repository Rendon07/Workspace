¿Ques es un framework CSS en diseño web?
Un framework CSS es una biblioteca de hojas de estilo que permite a los desarrolladores crear interfaces de usuario más rápido y con mayor compatibilidad. Evita empezar desde cero en cada proyecto y facilita la repetición y ajuste de diseños. Aunque puede ser complicado para principiantes, los desarrolladores con experiencia lo consideran muy útil.

¿Que es Bootstrap en diseño web?
Bootstrap es un framework web gratuito y de código abierto que facilita la creación de sitios responsivos y adaptados a móviles. Proporciona plantillas y componentes basados en HTML, CSS y JavaScript, permitiendo a los desarrolladores construir sitios más rápido sin preocuparse por funciones básicas.
